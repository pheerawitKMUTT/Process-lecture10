# Process Flow Diagram (PFD) for Sodium Hypochlorite Production

## Description
This document outlines the Process Flow Diagram (PFD) for the production of sodium hypochlorite (NaOCl). The PFD illustrates the key components, streams, and operating parameters involved in the sodium hypochlorite production process.

### Equipment Involved
1. **Electrolyzer**: Responsible for the electrolysis of sodium chloride (NaCl) solution.
2. **Cooling Tower**: Used to cool the outgoing streams from the electrolyzer.
3. **Reactor**: Where chlorine gas is reacted with sodium hydroxide (NaOH) to produce sodium hypochlorite.
4. **Storage Tank**: For storing the produced sodium hypochlorite solution.

### Key Streams and Parameters
- **Stream A**: Sodium Chloride Solution (Entering Electrolyzer) - Flow Rate: 100 L/h
- **Stream B**: Chlorine Gas (From Electrolyzer) - Flow Rate: 50 L/h
- **Stream C**: Sodium Hydroxide (Entering Reactor) - Flow Rate: 30 L/h
- **Stream D**: Sodium Hypochlorite Solution (Product) - Flow Rate: 70 L/h
